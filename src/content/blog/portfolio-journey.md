# 自分だけのポートフォリオを作る理由

エンジニアとして、自分のスキルや実績をアピールするためのポートフォリオは欠かせません。既存のテンプレートを使うのも良いですが、自分で一から作ることで、技術力の証明にもなります。

## 技術スタックの選定

今回は以下の技術を選定しました。

- **Next.js (App Router)**: 最新のReactフレームワーク。SEOに強く、パフォーマンスも高い。
- **Tailwind CSS v4**: スタイリングを効率化。v4のアルファ版（または最新版）を試すことで、最新トレンドへの感度もアピール。
- **Framer Motion**: 宣言的なアニメーションライブラリ。リッチな体験には必須。

## こだわったポイント

### 1. 3Dパララックス効果

Heroセクションには、マウスの動きに連動する3D視差効果を取り入れました。これにより、静的なサイトにはない没入感を生み出しています。

```typescript
const x = (mousePos.x - window.innerWidth / 2) / 50 * factor;
const y = (mousePos.y - window.innerHeight / 2) / 50 * factor;
```

### 2. モバイルレスポンシブ

PCだけでなく、スマホでも快適に見られるように細かく調整しました。特に、グリッドレイアウトを縦積みに変更したり、不要な装飾を非表示にするなどの工夫をしています。

## まとめ

ポートフォリオ作りは、新しい技術を試す最高の実験場です。もし迷っているなら、まずは簡単なものから作り始めてみましょう！
