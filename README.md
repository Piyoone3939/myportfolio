# Piyoone's Portfolio 🚀

学生エンジニア **PIYOONE** の個人的なポートフォリオウェブサイトです。
最新のウェブ技術を駆使し、**「没入感」**と**「使いやすさ」**を両立させたデザインを目指しました。

![Portfolio Preview](/public/projects/project1.jpg)
*(※実際のアプリのスクリーンショットに置き換えてください)*

## ✨ Features (特徴)

このポートフォリオは単なる実績紹介リストではありません。訪問者が楽しめるインタラクティブな体験を提供します。

- **🎨 Immersive Hero Section**:
  - マウスの動きに反応する **3Dパララックス効果**
  - 現在時刻を表示するリアルタイムクロック
  - 背景に流れるバイナリコード演出

- **🚀 Smooth Experience**:
  - **Framer Motion** によるシームレスな画面遷移とスクロールアニメーション
  - 直感的なマイクロインタラクションとホバーエフェクト

- **📱 Fully Responsive**:
  - デスクトップからモバイルまで最適化されたレスポンシブデザイン
  - モバイル端末でも情報の可読性を損なわないUI調整

- **📂 Hybrid Navigation**:
  - スクロールで概要を把握できるLPスタイル
  - 詳細をじっくり見れる個別ページ構成のハイブリッド設計

## 🛠 Tech Stack (技術スタック)

パフォーマンスと開発体験を重視したモダンな構成です。

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏁 Getting Started (セットアップ)

ローカル環境で起動する手順です。

### 前提条件
- Node.js (v18以上推奨)
- npm

### インストールと起動

```bash
# リポジトリのクローン
git clone https://github.com/Piyoone3939/MyPortfolio.git

# ディレクトリ移動
cd MyPortfolio

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。
